## 规则限制：

- 可作为角编的商品上架，为提升商品的预览效果，建议制作商品缩略图。[商品icon制作教程](./11_0_0_Product-icon-production)
- 上架商品需要制作对应的Icon
- 皮肤部分必须采用官方提供的基础模型，且**严禁对皮肤部分的** **UV** **进行修改**。
- 制作的模型需要匹配基础体型，**不得对体型比例进行任何修改。**
- **删除服装覆盖的裸模模型**
- 需要分部件制作，分别为上装，下装，手套，鞋子
